# Backend Baixinho Preços

API simples em Node.js + Express.

## Rotas

### GET /search
Parâmetros:
- q: termo do produto
- cep: localização

Retorna lista mockada de preços.

## Como rodar localmente
npm install
npm start

Servidor inicia em:
http://localhost:3000

## Deploy recomendado
- Render.com
- Railway.app
- Fly.io
